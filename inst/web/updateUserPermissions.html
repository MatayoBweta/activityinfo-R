Updates a user's permission to access a database.

Usage

updateUserPermissions(databaseId, userEmail, userName, partner, allowView = TRUE, allowViewAll = FALSE, allowEdit = FALSE, allowEditAll = FALSE, allowDesign = FALSE, allowManageUsers = FALSE, allowManageAllUsers = FALSE)

Arguments

databaseId
the id of the database
userEmail
the user's email address
userName
the user's full name. If the user with this email address already has an activityinfo account, this parameter will have no effect.
partner
the id or name of the partner to which the user will be added.
allowView
if TRUE, the user can view sites belonging to their partner
allowViewAll
if TRUE, the user can view sites belonging to all partners
allowEdit
if TRUE, the user can create and edit sites belonging to their partner
allowEditAll
if TRUE, the user can create and edit sites belonging to all partners
allowDesign
if TRUE, the user can create and modify the structure of the database
allowManageUsers
if TRUE, the user can change the permission of users for the database within their own partner
allowManageUsers
if TRUE, the user can change the permission of all users within the database

Description

If the user identified by the given email address does not yet have an ActivityInfo account, they will receive an email inviting them to create a new account and choose a password.
