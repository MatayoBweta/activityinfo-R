Creates a new site

Usage

createLocation(id = generateId(), locationTypeId, name, axe = NULL, longitude = NULL, latitude = NULL)

Arguments

id
the id for the new location
locationTypeId
the id of the locationType to which to add this location
name
the name of the location
axe
an optional secondary name of the location (originally axe routiere, but can be used for PCODEs or alternate spellings)
longitude
optional latitude of the location
latitude
optional longitude of the location

Description

Creates a new site
