getSitesDataFrame

Usage

getSitesDataFrame(activity)

Arguments

activity
an activity object
month
the month for which to fetch indicator values, in the format '2014-02'

Description

Fetch the sites for an activity as a data.frame

Details

If the activity has monthly reporting, indicator values will only be fetched if the month is specified
