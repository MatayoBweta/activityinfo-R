Adds a partner to a database.

Usage

addPartner(databaseId, partnerName, fullPartnerName = NULL)

Arguments

databaseId
the numeric id of the database to which the partner will be added
partnerName
the partner's acronym or shortened name
partnerFullName
the partner's complete name. Only used if a partner identified by 'partnerName' does not yet exist

Value

the id of the partner

Description

Currently, partners are globally shared objects with an identity derived from their name.
