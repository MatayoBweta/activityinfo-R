Creates a new Indicator

Usage

createIndicator(activityId, name, category = NULL, listHeader = NULL, description = NULL, units, aggregation = c("sum", "mean", "count"), sortOrder = NULL)

Arguments

activityId
the id of the activity to which this indicator should be added
name
the name of this indicator
category
the activity's category
listHeader
a short label to be used when displaying the indicator as a column
description
an extended description of this indicator
units
the units of measure of this indicator
aggregation
the method to use when aggregating this indicator's values ("sum", "mean", "count")
sortOrder
an ordinal value indicating the order of this indicator

Value

the id of the new indicator

Description

Creates a new Indicator
